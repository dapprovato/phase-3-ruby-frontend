# Animal Crossing Villager List

## Project Description:

### Allows users to

   *Villagers:*
   
        - create a new villager
        - view a list of all villgers
        - update which village that a villager lives in
        
   *Villages:*
   
        - view all villages
        - create a new village
        - check all villagers that currently live in a village
