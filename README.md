# Animal Crossing Villager List

## Project Description:

### Allows users to

   *Villagers:*
   
        - create a new villager : fill in the information for the villager name and the photo url, hit submit, refresh the page to see the new villager
        - view a list of all villgers : scroll to view all created villagers
        - update village for a villager : select drop down menu, select the new town you would like the villager moved to, press 'move villager' button
        - delete villager : select the 'remove villager' button to delete a villager from the list
